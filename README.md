## ✨ 项目介绍

**CameraPet** 是一款创新的虚拟宠物应用，它能够在宠物离开这个世界后，继续以虚拟宠物的身份，在电脑副屏继续陪伴主人：

- 📸 **根据你的照片生成宠物** —— 上传你拍摄的宠物照片，AI 会为你生成超高还原度的虚拟宠物
- 可以设定宠物种类，让ai模型来计算、模拟合理的行为举止。（支持接入本地模型或使用apikey）//此功能可能要待后续实现
- 🏠 **创建生活场景** —— 为你的宠物打造全新或跟照片无异的生活环境、支持多场景切换（就像家里多个摄像头，随意切换观察场景）
- 📹 **摄像头视角观察** —— 像用摄像头观察家里宠物一样，随时查看宠物的状态
- 🎤 **语音互动** —— 内置麦克风支持，虚拟宠物能听到主人的呼唤并作出回应

📦 安装

### 下载安装包

前往 [Releases](https://github.com/yourusername/CameraPet/releases) 页面下载最新版本。

### 从源码构建

```bash
# 克隆仓库
git clone https://github.com/yourusername/CameraPet.git
cd CameraPet

# 安装依赖
npm install

# 启动开发环境
npm run dev

# 构建应用
npm run build
```
