## ✨ 项目介绍

**CameraPet** 是一款创新的虚拟宠物应用。在宠物离开我们之后，它仍能以虚拟形象继续陪伴在你身边：

- 📸 **AI 生成虚拟宠物** —— 上传你拍摄的多张宠物照片或视频，AI 会全方位学习，生成超高还原度的虚拟宠物
- 🔊 **声音模拟**（规划中）—— 学习并模拟宠物的声音，让虚拟宠物不仅形似，声音也能以假乱真
- 🐾 **行为模拟**（规划中）—— 设定宠物种类，让 AI 模拟合理的行为举止（支持本地模型或 API Key）
- 🏠 **生活场景**（规划中）—— 为宠物打造真实或梦幻的生活环境，支持多场景自由切换
- 🎤 **语音互动**（规划中）—— 虚拟宠物能听到你的呼唤并作出回应
